# Security Policy

## Supported Versions

We provide security fixes for the latest release on the `main` branch only.

| Version | Supported |
| ------- | --------- |
| latest  | Yes       |
| older   | No        |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report security issues by emailing [security@nexuslabs.dev](mailto:security@nexuslabs.dev).

Include as much of the following as you can:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- Affected versions or commits
- Any suggested mitigations

We will acknowledge your report within **48 hours** and aim to release a fix within **14 days** for critical issues. We will keep you updated throughout the process and credit you in the release notes unless you prefer to remain anonymous.
